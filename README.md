# dotfiles

## Instalación
Lanza los siguientes comandos:
```bash
git clone https://github.com/sgcharameli/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install
```

## Configuración local GIT
En el fichero .gitconfig_local se debe añadir la configuracion para los commits:
```
[user]
  email = <EMAIL>
  name = <NOMBRE_USUARIO>
```
